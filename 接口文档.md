# GoTik 当前接口文档

## 说明

本文档基于当前项目代码整理，优先覆盖已经完成并用于联调的主要业务接口。

当前已整理的接口包括 `account` 模块、`video` 模块、`like` 模块、`comment` 模块与 `social` 模块，共 19 个：

1. `POST /account/register`
2. `POST /account/login`
3. `POST /account/findByID`
4. `POST /video/publish`
5. `POST /video/listByAuthorID`
6. `POST /video/getDetail`
7. `POST /video/uploadVideo`
8. `POST /video/uploadCover`
9. `POST /like/like`
10. `POST /like/unlike`
11. `POST /like/isLiked`
12. `POST /like/listMyLikedVideos`
13. `POST /comment/listAll`
14. `POST /comment/publish`
15. `POST /comment/delete`
16. `POST /social/follow`
17. `POST /social/unfollow`
18. `POST /social/getAllFollowers`
19. `POST /social/getAllVloggers`

说明：

1. 除上传接口使用 `multipart/form-data` 外，其余接口当前均为 JSON 请求与 JSON 响应
2. 当前还有一个已跑通但主要用于鉴权验证的接口：`GET /account/me`

## 1. 注册账号

### 接口地址

```text
POST /account/register
```

### 接口说明

创建一个新的账号。

### 请求体

```json
{
  "username": "testuser",
  "password": "123456"
}
```

### 请求字段说明

| 字段 | 类型 | 必填 | 说明 |
| ---- | ---- | ---- | ---- |
| username | string | 是 | 用户名 |
| password | string | 是 | 密码 |

### 成功响应

```json
{
  "message": "account created"
}
```

### 失败响应

#### 参数绑定失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`400`

#### 创建账号失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`500`

```text

```

### 接口说明

发布一个新视频。该接口需要登录后访问，请在请求头中携带合法 token。


## 7. 上传视频文件

### 接口地址

```text
POST /video/uploadVideo
```

### 接口说明

上传视频文件并返回可访问的播放地址。该接口需要登录后访问，请在请求头中携带合法 token。

### 请求头

```text
Authorization: Bearer your_jwt_token
Content-Type: multipart/form-data
```

### 表单字段

| 字段 | 类型 | 必填 | 说明 |
| ---- | ---- | ---- | ---- |
| file | file | 是 | 仅支持 `.mp4`，最大 `200MB` |

### 成功响应

```json
{
  "url": "http://localhost:8080/static/videos/1/20260312/abc.mp4",
  "play_url": "http://localhost:8080/static/videos/1/20260312/abc.mp4"
}
```

### 失败响应

#### 未登录、缺少文件或文件不合法

```json
{
  "error": "具体错误信息"
}
```

状态码：`400`

#### 保存失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`500`

## 测试建议

当前建议按以下顺序测试接口：

1. 先调用 `/account/register` 注册一个新用户
2. 再调用 `/account/login` 获取 token
3. 调用 `/account/findByID` 查询用户信息
4. 使用登录得到的 token 调用 `/video/uploadVideo` 和 `/video/uploadCover` 获取上传地址
5. 使用上传后的地址调用 `/video/publish` 发布视频
6. 调用 `/video/listByAuthorID` 查询某作者的视频列表
7. 调用 `/video/getDetail` 查询视频详情
8. 使用登录得到的 token 调用 `/like/like` 点赞视频
9. 调用 `/like/isLiked` 检查点赞状态
10. 调用 `/like/listMyLikedVideos` 查看当前用户点赞过的视频
11. 调用 `/like/unlike` 取消点赞后再次检查状态
12. 使用登录得到的 token 调用 `/comment/publish` 发布评论
13. 调用 `/comment/listAll` 查看某视频下的评论列表
14. 调用 `/comment/delete` 删除评论并再次检查列表变化
15. 使用登录得到的 token 调用 `/social/follow` 关注目标用户
16. 调用 `/social/getAllVloggers` 查看自己的关注列表
17. 调用 `/social/getAllFollowers` 查看目标用户粉丝列表
18. 调用 `/social/unfollow` 取消关注后再次检查列表变化
