# GoTik 当前接口文档

## 说明

本文档基于当前项目代码自动整理，和目前实际注册的路由保持一致。

当前已开放的接口包括 `account` 模块、`video` 模块与 `like` 模块，共 10 个：

1. `POST /account/register`
2. `POST /account/login`
3. `POST /account/findByID`
4. `POST /video/publish`
5. `POST /video/listByAuthorID`
6. `POST /video/getDetail`
7. `POST /like/like`
8. `POST /like/unlike`
9. `POST /like/isLiked`
10. `POST /like/listMyLikedVideos`

目前接口均为 JSON 请求与 JSON 响应。

## 1. 注册账号

### 接口地址

```text
POST /account/register
```

### 接口说明

创建一个新的账号。

### 请求体

```json
{
  "username": "testuser",
  "password": "123456"
}
```

### 请求字段说明

| 字段 | 类型 | 必填 | 说明 |
| ---- | ---- | ---- | ---- |
| username | string | 是 | 用户名 |
| password | string | 是 | 密码 |

### 成功响应

```json
{
  "message": "account created"
}
```

### 失败响应

#### 参数绑定失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`400`

#### 创建账号失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`500`

## 4. 发布视频

### 接口地址

```text
POST /video/publish
```

### 接口说明

发布一个新视频。该接口需要登录后访问，请在请求头中携带合法 token。

### 请求头

```text
Authorization: Bearer your_jwt_token
```

### 请求体

```json
{
  "title": "我的第一个视频",
  "description": "测试视频",
  "play_url": "http://example.com/video.mp4",
  "cover_url": "http://example.com/cover.jpg"
}
```

### 请求字段说明

| 字段 | 类型 | 必填 | 说明 |
| ---- | ---- | ---- | ---- |
| title | string | 是 | 视频标题 |
| description | string | 否 | 视频描述 |
| play_url | string | 是 | 视频播放地址 |
| cover_url | string | 是 | 视频封面地址 |

### 成功响应

```json
{
  "id": 1,
  "author_id": 1,
  "username": "testuser",
  "title": "我的第一个视频",
  "description": "测试视频",
  "play_url": "http://example.com/video.mp4",
  "cover_url": "http://example.com/cover.jpg",
  "create_time": "2026-03-10T00:00:00Z",
  "likes_count": 0,
  "popularity": 0
}
```

### 失败响应

#### 参数绑定失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`400`

#### 未登录或 token 非法

```json
{
  "error": "具体错误信息"
}
```

状态码：`401`

#### 发布失败

```json
{
  "error": "具体错误信息"
}
```

状态码：`400` 或 `500`

## 测试建议

当前建议按以下顺序测试接口：

1. 先调用 `/account/register` 注册一个新用户
2. 再调用 `/account/login` 获取 token
3. 调用 `/account/findByID` 查询用户信息
4. 使用登录得到的 token 调用 `/video/publish` 发布视频
5. 调用 `/video/listByAuthorID` 查询某作者的视频列表
6. 调用 `/video/getDetail` 查询视频详情
7. 使用登录得到的 token 调用 `/like/like` 点赞视频
8. 调用 `/like/isLiked` 检查点赞状态
9. 调用 `/like/listMyLikedVideos` 查看当前用户点赞过的视频
10. 调用 `/like/unlike` 取消点赞后再次检查状态
